# 🕋 Dawah Center CRM & Management System
A professional, bilingual (Arabic/English) Customer Relationship Management (CRM) system designed for the Association Dawah Center. Optimized for mobile deployment on **Infinix Xpad** tablets.

## ✨ Key Features
* **Bilingual UI:** Fully localized interface with RTL (Arabic) and LTR (English) support.
* **Mass Data Export:** One-click Excel/CSV generation for supervisor reporting.
* **WhatsApp Integration:** Integrated tracking system to log the last time a visitor was contacted.
* **Smart Dashboard:** Real-time record counter with smooth CSS animations for instant feedback.
* **Architecture:** Client-Server model running on XAMPP (Port 3307) for local area network access.

## 🛠️ Technical Stack
* **Backend:** PHP 8.2
* **Database:** MySQL (MariaDB)
* **Frontend:** HTML5, CSS3 (Flexbox/Animations), JavaScript
* **Deployment:** Local Server Hosting via Apache

## 📱 Tablet Optimization
Specifically configured for the **Infinix Xpad (Helio G99)** to ensure high performance and fluid transitions using the tablet's 90Hz display.
